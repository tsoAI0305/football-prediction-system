# 🤝 貢獻指南

感謝你對 AI 足球預測系統的興趣！

## 📋 如何貢獻

### 🐛 報告 Bug
前往 Issues 頁面描述問題

### 💡 建議新功能
開啟 Issue 並標註 enhancement

### 🔧 提交程式碼
1. Fork 專案
2. 建立分支: git checkout -b feature/your-feature
3. 提交修改: git commit -m "feat: 描述"
4. 推送並開 PR

## 🛠️ 開發環境

### 後端
cd backend && python -m venv venv && pip install -r requirements.txt

### 前端
cd frontend && npm install && npm run dev

## 📝 提交訊息規範
feat: 新功能
fix: 修復
docs: 文檔
style: 格式
refactor: 重構
test: 測試
